# AI Long-Term Memory System

## Kaynak
Bu dosya, bir AI asistanına uzun vadeli hafıza sistemi kurmak için kullanılan bir rehberden alınmıştır.

## 5 Ana Bölüm

### 1. Explicit Triggers (Açık Tetikleyiciler)
İnsanın "hatırla", "preferansım şu", "not al" gibi doğrudan söylediği durumlar.

### 2. Implicit Triggers (Örtük Tetikleyiciler)
İnsan söylemese bile AI'ın otomatik hatırlaması gereken durumlar:
- Kullanıcı hakkında gerçekler
- Kurulan workflow'lar
- Başarısızlıklardan öğrenilen dersler

### 3. Memory File Structure (MEMORY.md)
Kategoriler:
- Identity (Kimlik)
- Preferences (Preferanslar)
- Decisions (Kararlar)
- Projects (Projeler)
- People (Kişiler)

### 4. Rules (Kurallar)
- Her girdiye tarih ekle
- Her gerçeği tek satırda tut (atomic)
- Hassas veri (şifre, API key) saklama
- Düzenli temizlik yap

### 5. Self-Review (Öz Denetim)
- Periyodik olarak tekrarları kaldır
- Çelişkileri çöz
- Eski bilgileri arşivle
